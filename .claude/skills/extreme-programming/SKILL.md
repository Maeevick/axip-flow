# Extreme Programming

## Purpose

Non-negotiable engineering culture. Every implementing agent loads this. No exceptions.

## Routing Table

| Practice | Reference | Load When |
|----------|-----------|-----------|
| TDD / Test First | `references/tdd-test-first.md` | Writing any code — always |

## Core Rules (no reference needed)

- Small, verifiable increments. No big-bang delivery.
- CI must pass before any task is considered done.
- If it is not tested, it does not exist.
- If it needs a comment, it needs a rename.
- Discovery before implementation. Read the project before touching it.
